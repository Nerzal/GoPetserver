# GoPetserver
Repository to learn golang using a petserver

# GO Basics

## General
 - Composition over inheritance
 	- There is no inheritance
 - No Method Overloading
 	- Each Method is unique
 - Go erzwingt Egypt-Style Klammern
 - https://golang.org
 - IDE VSCode / Goland
## Visibility
   - lowercase = private
   - uppercase = public

## Packages
 - Similar to namespaces
 - Import
 	- Imports Packages
 		- Local
 		- Remote (github repo)
 		- SourceCode
## Types
### Basic
 - int
 - bool
 - string
 - uint, uinptr
 - float32, float64
 - complex64, complex128
 - byte (uint8)
 - rune (char)

### Advanced (complex)
 - Array
 	- numbered collection
 - Slice
 	- array that can grow in size (like a list)
 - Struct
 - Pointer
 - Function
 - Interface
 	- Not implemented explicitly
 - Map
 	- Like a dictionary
 - Channel
 	- Used for communication between goroutines
